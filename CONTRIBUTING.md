# Contributing

Thanks for helping improve this project!

## Quick start
```bash
npm install
npm run check
```

## Development notes
- Keep files under ~400 lines. Prefer small, focused modules.
- Follow the existing folder layout in `src/` and add docs for any new tool groups.
- Add tests for new behavior, especially where it is deterministic and low-effort.

## Commands
- `npm run lint`
- `npm run typecheck`
- `npm test`
- `npm run test:coverage`

## Tooling
This repo uses TypeScript, ESLint, Prettier, and Vitest. Format and lint changes should be clean and consistent.
